# Document 02: Slow Computer Performance
## 1. Overview

A slow computer is one of the most common issues reported to IT support teams. Users may experience:

* Slow startup and login times
* Applications taking a long time to open
* System freezing or becoming unresponsive
* Delayed keyboard and mouse response
* Slow file access
* Poor performance during video meetings

Performance troubleshooting requires a structured approach to determine whether the issue is caused by:

* Hardware limitations
* Resource-intensive applications
* Windows configuration
* Malware/security threats
* Storage problems
* User profile corruption
* Network dependency issues

---

# 2. Problem Description

## Example User Ticket

> "My laptop has become very slow. It takes 10 minutes to start, applications freeze frequently, and Teams meetings are lagging."

# 4. Common Causes

## Hardware Issues

* Insufficient RAM
* Slow HDD instead of SSD
* Overheating CPU
* Aging hardware
* Low disk capacity

---

## Software Issues

* Too many startup applications
* Background processes consuming resources
* Corrupted Windows files
* Unnecessary services
* Pending Windows updates
* Malware infections

---

## User Environment Issues

* Large browser cache
* Large Outlook mailbox cache
* Excessive files on desktop
* Corrupted user profile

# 14. PowerShell Cleanup Script

Create:

```
Scripts/Cleanup-Temporary-Files.ps1
```

Content:

```powershell
Write-Host "Cleaning temporary files..."

Remove-Item `
"C:\Windows\Temp\*" `
-Recurse `
-Force `
-ErrorAction SilentlyContinue


Remove-Item `
"$env:TEMP\*" `
-Recurse `
-Force `
-ErrorAction SilentlyContinue


Write-Host "Cleanup completed"
```

---

## Script Explanation

### Remove Windows temporary files

```powershell
C:\Windows\Temp
```

Removes system temporary files.

---

### Remove User temporary files

```powershell
$env:TEMP
```

Removes temporary files created by applications.

---

### Parameters

| Parameter                     | Purpose                      |
| ----------------------------- | ---------------------------- |
| -Recurse                      | Removes folders and contents |
| -Force                        | Removes hidden files         |
| -ErrorAction SilentlyContinue | Ignores locked files         |

# 16. Repair Windows System Files

Performance problems can result from corrupted files.

Run:

```cmd
sfc /scannow
```

---

Then:

```cmd
DISM /Online /Cleanup-Image /RestoreHealth
```

# 19. Create System Health Report Script

Create:

```
Scripts/System-Health-Check.ps1
```

Script:

```powershell
Write-Host "System Health Report"

Write-Host "CPU Information"
Get-CimInstance Win32_Processor |
Select Name, LoadPercentage


Write-Host "Memory Information"

Get-CimInstance Win32_OperatingSystem |
Select TotalVisibleMemorySize, FreePhysicalMemory


Write-Host "Disk Information"

Get-PhysicalDisk |
Select FriendlyName, HealthStatus
```

---

## Purpose

Provides quick health information:

* CPU load
* Available memory
* Disk health

Useful for:

* Remote support
* Ticket investigation
* Device audits

# 22. Lesson Learned

* Always identify the root cause before applying fixes.
* Task Manager provides the first indication of resource problems.
* Performance issues can be caused by software, hardware, or user behaviour.
* PowerShell automation improves troubleshooting efficiency.
* Regular maintenance prevents recurring performance problems.
* Documentation helps create repeatable troubleshooting processes.

---

# 23. Preventive Best Practices

## Users

* Restart computers regularly.
* Keep storage above 20% free space.
* Avoid unnecessary startup applications.
* Install approved software only.

## IT Administrators

* Monitor device health using Intune.
* Replace failing HDDs with SSDs.
* Maintain Windows update compliance.
* Review application deployment impact.

---

# 24. Escalation Criteria

Escalate when:

* Hardware replacement is required.
* Disk health reports failure.
* Performance issue affects multiple users.
* Malware infection is suspected.
* Windows corruption cannot be repaired.
